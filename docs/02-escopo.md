# Escopo do Projeto

**Versão:** 1.0
**Data:** 16/07/2026
**Status:** Aprovado

## Objetivo do Sistema

O objetivo do sistema é facilitar o processo de compra do cliente.

## Objetivos específicos

1. Reduzir o tempo de interação do cliente com o proprietário da doceria.
2. Facilitar o recebimento de pedidos da loja.
3. Organizar os pedidos com facilidade para o proprietário da doceria.
4. Simplificar a criação de pedidos do cliente.

## Metas do sistema

1. Pedidos do cliente finalizados com mais precisão e velocidade.
2. Proprietário da doceria recebendo os pedidos de forma organizada, completa e padronizada.
3. Redução de informações desnecessárias durante o processo de pedido.
    
## Resultado esperado para o sistema
    
De acordo com os objetivos informados acima, o proprietário da doceria conseguirá focar mais na produção e menos no atendimento inicial aos usuários.
