# Glossário

## Cliente

Pessoa que acessa o site para consultar produtos e realizar pedidos.

## Proprietário da doceria

Pessoa responsável pela administração da loja e contratante do sistema. Também é quem recebe os pedidos enviados pelo site.

## Pedido

Conjunto de produtos selecionados pelo cliente antes do envio.

## Carrinho

Funcionalidade responsável por armazenar temporariamente os produtos escolhidos pelo cliente.

## WhatsApp

Canal utilizado para o envio dos pedidos ao proprietário da doceria.
